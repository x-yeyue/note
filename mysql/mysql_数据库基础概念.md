# 信息与数据

> **信息 = 数据 + 语义**

## 信息(Information)

- *概念*: 对人类有用的事实和知识的集合。
- *定义*: 用文字、数字、符号、图像等介质来表示事件、物质、现象的内容、数量和特征，向人们提供关于现实世界新的事实和知识。作为生产、经营、管理、分析和决策的依据。
- *特性*：传播性、共享性、试用性、时效性等特性。

## 数据(Data)

- *概念*: 数据库中存储的基本对象。
- *定义*: 信息的载体，是信息的一种符号化表示，描述事物的符号记录。
- *种类*: 文本、图形、图像、音频、视频、学生的档案记录、货物的运输情况等。
- *特点*: 数据与其语义是不可分的。
- *含义*: 数据的语义。数据与其语义是不可分的。

**例：**
	*数据*: (0005796, 518, 张三, 1, 1999.1.1, 01)
	*语义*: 工号，部门编号，姓名，性别，出生日期，民族
	*解释*: 教师张三，男，1999.1.1出生，汉族，工号0005796，在软件教研室办公

# 数据库

## 数据库(Database, DB)

- 数据的仓库，数据具有相关性。
- 长期保存在计算机内、有组织、可共享的、大量数据的集合。
- 源于对规范化“表”(Table) 的处理。
